Konstruktor tworzy góre z ostrym szczytem.

Funkcja tworzy góre z ostrym szczytem, obliczając nowe współrzędne na podstawie współrzędnych z pliku wzorcowego i zapisuje je do pliku roboczego.

Parameters
[in]posWektor położenia początkowego (definiowany jako wektor przesunięcia wzorcowego prostopadłościanu)
[in]scaleWektor skali figury.
Returns
Zwraca true jeśli udało się otworzyć pliki i stworzyć płaskowyż lub false jeśli operacja się nie powiodła.
